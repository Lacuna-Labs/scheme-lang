Try it

The REPL below is the real interpreter, bundled for the browser. Everything you type is evaluated locally — nothing leaves your device. Tab to complete, comma to run a REPL command, Enter to evaluate.

Some things to try: 

(+ 1 2 3 4 5)
(map (lambda (x) (* x x)) '(1 2 3 4 5))
(circle 50 50 30)
(define (fact n) (if (< n 2) 1 (* n (fact (- n 1)))))
(fact 10)

Or install locally: 

curl -fsSL https://raw.githubusercontent.com/Lacuna-Labs/scheme-lang/main/install.sh | sh

Make your own dialect

scheme-lang is the base — no product verbs. Fork it, add your own vocabulary, and every user gets the same “Try it →” affordance on their own Pages site for free. See TEMPLATE-FOR-FORKS-PAGES.md for the pattern.
